# 六爻占卜 · 六套铜钱皮肤

> **图案为主、文字为辅** — 服务于三枚铜钱起卦：字面（阳）纹样饱满可辨，背面（阴）纹样克制且与字面明显不同，便于掷出后识读 2/3。

## 设计逻辑

| 币 | 占卜意象 | 字面（阳）主纹 | 背面（阴）主纹 | 点睛小字 |
|----|----------|----------------|----------------|----------|
| 半两 | **起卦** | 四出文（方孔射廓） | 三断爻（阴爻） | 卦 |
| 五铢 | **五运** | 五辐射 + 四出 | 单实爻 + 淡五角 | 爻 |
| 大泉五十 | **变革** | 十字泉脉 + 八角辐 | 空心十字 + 双断爻 | 变 |
| 开元通宝 | **流通** | 先天八卦环 | 洛书九宫点 | 易 |
| 大观通宝 | **观照** | 卷云 + 三实爻 | 三断爻 | 觀 |
| 洪武通宝 | **阴阳** | 日月合璧 + 十二辐 | 单月（阴） | 陰 |

小字位于廓下居中，约 30–36px，用 `RELIEF_DIM` 半透明，不抢主纹。

## 模块

| 文件 | 职责 |
|------|------|
| `oracle_patterns.py` | 纯 SVG 纹样（四出、八卦、洛书…） |
| `oracle_skins.py` | 六币主题 + 方孔形制 + 点睛字 |
| `bronze_face.py` | 统一暗铜质感 |
| `gen-type-skins.py` | 烘焙 |

## 命令

```bash
python3 ../fonts/setup-fonts.py
python3 gen-type-skins.py
```

## 与起卦规则

- App 以字面为阳（3）、背面为阴（2）；两面图案差异是第一识读线索，纹理亮度为第二线索。
- 默认素钱仍用 `back-su-yang` 四出文逻辑，与本套「半两·起卦」一脉相承。
