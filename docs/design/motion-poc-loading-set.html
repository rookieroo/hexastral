● fate-app · Loading Concepts

Loading 概念集 · 七选一

不靠"大曲线波浪"。前五个是水墨/金石的静(笔/墨/卦/枢/息);后两个 F·G 是回应你的反馈 —— 墨蚀=月影"水墨动态渲染"(边界一直在流、在吞),吞噬=细胞吞噬式 metaball 融并。全部循环、可慢放对比。选中后接进 fate-flow 的「推演中」。

推演中
一笔圆相Ensō
一笔起、一笔收,留飞白缺口。落墨成圆 = 一次推演收束。最静、最"水墨"。
推演中
墨滴入水Ink-bloom
墨点入纸,洇开又散,一朱一墨交替。物理是"扩散",不是描线。
起卦中
六爻成卦Casting
自下而上,一爻一爻落定成卦。最贴 HexAstral 之名,叙事感最强。
推演中
星枢旋Astrolabe
三环异速回转 + 朱枢、金标。罗盘/星盘味,金石感最足。
推演中
墨息Ink-breath
一点墨珠起伏吐纳,淡晕一圈圈漫出。极简、近"呼吸",久看不腻。
推演中
墨蚀Ink-eclipse
月面上墨影活渲染:边界一直在流、在吞,非滑动硬影。← 你说的"阴影=水墨动态"。
推演中
吞噬Engulf
墨核吞食四围墨点,液态融并(metaball)。← 你说的"细胞吞噬",生命感最强。
共同原则:① 一律 朱 / 金 / 骨 三色,深石底;② 动得慢而笃定(2.8–3.8s 一循环),不炫技;③ 纹理靠 feTurbulence 小位移做"手感",非大波浪;④ 都能落进圆形/方形遮罩。
取舍提示:一笔圆相最雅最通用;六爻成卦最扣品牌、最有"在推演"的叙事;星枢旋最像传统命理器物;墨息/墨滴最轻、最适合频繁短 loading。你定一个,或"主用 X + 短场景用 Y"。
