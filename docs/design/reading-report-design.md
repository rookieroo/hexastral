# HexAstral · Reading Report 设计（命书内容 / Free·Full / 导航）

> 目标：reading report 的**字段与结论**都对得上**真实八字盘 / 紫微盘**能读出的东西（不杜撰），
> 再切成 **Free（浅尝辄止）** 与 **Full**，并定一套**导航架构**承载"一份很长的命书"。
> 视觉/动效复用 [motion-asset-system.md](motion-asset-system.md)（纸/墨/朱 + 命盘绘制 + 墨晕转场 + 盖印）。

## 1. 真实盘能读出什么（内容地基）

### 八字（BaZi · 四柱）
- 出生年月日时 → **四柱**：年/月/日/时 各一**天干 + 地支** = 八字。**日干 = 日主/日元 = 命主本人**。
- **五行力量**：把八字（含地支**藏干**）按五行加权——**月令(月支)权重最高**（定旺相休囚死），再调**合/冲/刑/害** → 得到**五行强弱分布** → **身强 / 身弱**判定。
- 由强弱定 **用神（喜·补平衡的那个五行）/ 忌神（损平衡的）**。→ 推**宜忌方位/颜色/行业/择时**。
- **十神**（比肩劫财·食神伤官·正偏财·正官七杀·正偏印）分布 → **性格、六亲、事业、财、婚、健康**倾向。
- **大运（十年）/ 流年** → 吉凶起伏（命书的 timeline）；**格局**定整体层次。

### 紫微斗数（ZiWei）
- 出生时间 → **十二宫**固定排盘：命·兄弟·夫妻·子女·财帛·疾厄·迁移·交友(仆役)·官禄·田宅·福德·父母。
- **主星(14)+辅煞星** 落各宫；**四化（化禄/权/科/忌）** 由生年干触发 = 关键动力/转折。
- 紫微**不做五行打分**；它有 **五行局**（水二/木三/金四/土五/火六局），只用来定**大限**起运岁数（是分类，不是分数）。
- 读法：**命宫主星 → 命格/性格**；**各宫星曜 → 对应人生域**（夫妻宫→婚、财帛宫→财…）；四化→转折；大限/流年→时机。

### 合盘
- 经典"合盘" = **两人**合参（合婚/关系）→ 更适合 **yuan 緣 / 合婚**功能。
- 单人产品（fate 命）更值得做 **双系统交叉读**：八字 + 紫微对同一人**互相印证**（八字给"力量/喜忌"，紫微给"宫位/星性"），结论更立体、更"准"。

## 2. 「五行得分」怎么理解（含一处要修正）

- 它**是真东西**：= 八字的**五行力量分布**（哪行旺、哪行弱）→ 推**身强弱 → 用神/忌神**。所以放进报告**合理**。
- **但不是统一标准分**：各流派加权不同，没有"官方 32 分"。→ 产品里应表达为**相对分布 + 定性**（旺/相/休/囚/死 或 身强弱），而非假装精确分。
- **要修正的点**：**用神 ≠ 最旺**。用神是**补平衡**的那一行，命主**身强**时用神往往是**克泄耗**它的（弱的那几行），不是最旺那行。
  - 现 POC 里我把**最旺的木**标了朱（当令），容易被误读成"木是好的/用神"。
  - 改法：**两个语义分开标**——「**最旺**」（中性，墨色或细标）与「**用神**」（朱色高亮 + 一句"宜补X"）。把朱留给**用神**这个**行动指引**，更准也更贴"朱砂=指引"。

## 3. Reading Report 内容模型（真实输出 → 报告字段）

| 报告区块 | 来源 | 说明 |
|---|---|---|
| 本命盘（四柱/星盘图） | 八字四柱 + 紫微命宫 | "你的盘"，绘制动画(已做) |
| 命格定性 | 日主 + 命宫主星 | 一句话定性（身强弱 / 主星格） |
| 五行旺衰 | 八字五行分布 | 分布图 + **用神/忌神**（朱=用神） |
| 现运 / 今日 | 大运·流年 / 择日 | 当前段 + 宜忌（首屏今日签 + 命书顶） |
| 大运 timeline | 八字大运 | 你在哪段、还有哪些段(已做) |
| 各域详批 | 十神 / 紫微各宫 | 事业·财帛·姻缘·健康·六亲（→ L2 章） |
| 转折点 | 紫微四化 / 流年 | 关键年/事（择时） |
| 调候建议 | 用神 | 颜色/方位/行业/节律（行动指引，朱批） |

> 原则：**每个区块都能映射到真实算法输出**，照着真实盘排版 → 可信、不空。

## 4. Free（浅尝辄止） vs Full

**心法**：Free 要给**真实、准的一口**（证明价值），但**收住"可行动的深度 + 广度"**；不能空到像骗，也不能多到没理由付费。

| | Free（浅尝辄止） | Full |
|---|---|---|
| 本命盘 | ✅ 完整绘出（"你的盘"最该给，震撼） | ✅ |
| 命格定性 | ✅ 一句（身强弱 / 命宫主星格） | ✅ + 详解 |
| 五行旺衰 | ✅ 分布图可见；**用神/忌神 + 调候建议 锁** | ✅ 全开 + 宜忌行动项 |
| 现运/今日 | ✅ 一条 headline | ✅ + 逐项 + 流年 |
| 各域(事业/财/婚/健康/大运) | **封印**：标题 + 一句钩子(截断)，主体锁 | ✅ 逐章详批 |
| 紫微12宫 / 四化转折 | 锁 | ✅ |
| 合盘(关系) | 入口 | ✅ |

- **解锁动效** = 朱印「**启封**」/ 墨幕揭开（[motion §5-5](motion-asset-system.md)）：封印章一裂、墨退、隐藏命书显出 —— 变现时刻最扣母题。
- **控制度的尺子**：Free ≈ **1 屏真价值 + 可见但封印的深度**（让人看见"还有这么多"）；Full = 整本可滚的命书。封印区**可见标题/钩子**，比"灰一片"更勾人、也更诚实。

## 5. 导航架构（一份很长的命书怎么走）

真实 reading 很长（八字 4–6 域 + 大运流年；紫微 12 宫）→ **不可能一屏**。比较：

| 方案 | 评 |
|---|---|
| 全部一条竖滚 | 太长、无结构、回找某域难 |
| 卡片左右滑 | 适合**固定少量等量章**；深浅不一 + 不知有几张 → 差 |
| 翻页 | 拟物但移动端长文笨重 |
| **Hub 滚动 + 二级章页**（推荐） | 概览可扫 + 卷轴感；深内容按域分页、可深链/分享单章 |

**定稿三层：**
- **L0 首屏**：今日签 / 现运（日用入口）。→「展开命书」
- **L1 命书（竖向卷轴 = 我们刚做的可滚命书）= Hub/总览**：本命盘图 + 命格定性 + 五行/用神 + 现运 + **章目录**（事业·财·婚·健康·大运·紫微宫…，每行一句钩子 + 入口；Free 在此封印部分章）。**入场 = 墨晕转场 → 命盘绘制成形**（已做）。
- **L2 章详（二级页·按域）**：进某章 = **墨晕局部化开 / 纸笺抽出**；页内竖滚（该域的盘片段绘制 + 象/断/宜忌/择时 + 闲章落款）。**同级章之间左右滑**（事业↔财↔婚），不必回 Hub。

> 即：**L1 竖滚总览 → 点章 → L2 竖滚详页（L2 之间横滑）**。卷轴的"长"与分域的"清晰"兼得；章目录同时是 Free/Full 的**封印面**。
> 二级页设计：宣纸底 + 章名(闲章盖印) + 本域盘片段(绘制) + 读文(分段飞白揭示) + 落款；复用同一墨语言。

3. **L2 章详**：1 个样例域（如「财帛」）二级页 + L1→L2 墨晕转场 + L2 间横滑。
4. **五行修正**：用神/最旺分开标（朱=用神 + "宜补X"）。
