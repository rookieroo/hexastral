月相变体 · 单源 CSS 派生

从一张 ChatGPT 生成的残月 PNG 出发，通过 transform（旋转 / 镜像）+ filter（亮度 / 色温 / 对比度）+ 可选的 radial-gradient overlay（暗面加深）派生 8 个 app 的 logo 变体。 全部基于同一张源图，无 AI 二次生成。

物理限制：本方法只能改变高光朝向和整体明暗，不能凭空补出更多亮面。 源图是新月（亮面 ≈ 10%）→ 8 个变体全是「新月的不同朝向」，做不出残月 → 上弦 → 满月的亮面比例变化。 如需真正的月相递进（5% → 30% → 70%），必须回 ChatGPT 用不同 prompt 单独生成 8 张。
预览方法：把那张大预览的 PNG 命名为 moon-source.png 放进 docs/design/（与本文件同目录），然后用浏览器打开本文件。
如果接受单源派生：下一步我写一个 Node 脚本 tooling/icon-bake/derive.ts，用 sharp + svg-compositor 把这 8 套 transform/filter 烘焙成静态 PNG 1024 / 180 / 120 / 76 / 60 / 40 / 29 / 20 全套尺寸，输出到每个 app 的 assets/。

如果不接受（要真月相递进）：我写 8 套 ChatGPT prompt（每个相位一张，保持 sref/style 一致），你贴进 ChatGPT 各生成一张 1024×1024，覆盖到 8 个 app。
