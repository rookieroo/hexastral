水墨天体

在 V7 天体思路上加真质感:把光滑渐变球换成水墨绘的天体——浓淡 · 飞白 · 渗化 · 烘云托月 · 边缘不齐。 一举治好 V7 的两个病:没质感(墨色与笔触有厚度)、没文化(东方魂用"媒介"找回,而非靠汉字)。角落一枚朱砂钤印=画的落款,也是品牌记号。
下方第①②③为程序化近似(SVG 模拟墨韵)——只为看方向与氛围;真正成稿请走第④的插画 / AI 规格。

01

天体家族 · 水墨八象

同一轮月的八种天象,皆以墨写就。差异在相 / 蚀 / 环 / 双 / 山月 / 列宿,不靠颜色。
02

放大 · 墨韵

放大看墨的浓淡过渡与烘云——程序化只能近似到此;真水墨的飞白枯笔、绢丝渗化要手作。

烘云托月,
不画月而月自明。

水墨画月的正法是留白:月本身不着笔,反以墨渲染其周遭夜天,衬出一轮清辉。 本预览用 SVG 渐变 + 湍流位移近似墨色与不齐边缘;飞白 / 枯笔 / 绢本渗化是手绘的灵魂,代码够不着,见第④规格。

04

出图规格 · 交插画师 / AI（真正成稿）

插画 BRIEF

  • 媒介:水墨 / 工笔淡彩;两种地子可选 —— 深绢本(承接 V7 暗调)或 宣纸留白月(更正统)。
  • 母题:八 App = 同一轮月的天象 —— 满月 / 双月 / 环月 / 半相 / 月蚀 / 弦月 / 列宿(五星) / 山月。
  • 技法:浓淡分明、飞白枯笔、烘云托月(留白成月)、边缘不齐(忌正圆)、墨为主 + 一点朱砂钤印。
  • 规范:主体居中安全区、留白充分、可缩放;交付 1024 PNG + 矢量描边备份;朱砂印位置统一。

AI 提示词（Firefly 商用最稳 / 也可 SD·Flux / Midjourney）

满月 · 通用traditional Chinese ink wash painting, sumi-e full moon, soft tonal gradation, flying-white dry brush, wet bleeding edges, luminous moon on dark indigo silk, vast negative space, minimal, a small vermillion seal in corner, no text --ar 1:1
月蚀(六爻) / 弦月(解梦) / 山月(风水)…replace subject: "ink wash solar-eclipse ring / waning crescent with mist / a moon above misty ink mountains", keep style + dark silk + seal
