Firefly / SD · 外部位图 → 合成进印框

我不能在此生成位图，也不替你嵌未核实授权的图。真实路径有二：你用 Firefly/SD 生成（我给提示词），或从 CC0 机构取（认准具名机构）。 拿到 PNG 后，用下面的 mask 合成法把它"灌"进字形 —— 字成为透出纹理的窗口，与同一印框统一。右侧活例已用程序化石面演示该管线，把 rect 换成你的 image 即可。 建议：图标本体走程序化（16px 下真拓片只会糊成脏点）；真位图留给英雄图/营销页/启动图。

  <defs>
    <mask id="glyph">        <!-- 白=保留 -->
      <g style="color:#fff">
        { 你的字形 path / charGlyph }
      </g>
    </mask>
  </defs>

  <use href="#frame"/>

  <!-- 换成你的 Firefly/SD 位图 -->
  <image href="rubbing.png"
         width="100" height="100"
         preserveAspectRatio="xMidYMid slice"
         mask="url(#glyph)"/>
</svg>
