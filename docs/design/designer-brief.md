## Constraints we need you to navigate

### Visual constraints

1. **Family unity at App Store grid scale (60-80pt)** — when our 8 apps appear side by side in search results or a "more apps by this developer" tray, they must read as one brand
2. **Per-app differentiation at the same scale** — but the user must still distinguish `fate` from `yuan` from `cycle` instantly
3. **Premium feel** — apps are paid (some subscription). The icon should reflect serious craft. High contrast, refined material quality, no flat amateur look
4. **Cultural neutrality** — avoid explicit Chinese symbology (印章 seals, 毛笔 brushstrokes, 卷轴 scrolls, classical 篆体 characters as primary element). Tradition can be invoked subtly through proportion, palette, materiality — but not literal motifs. We're targeting US audiences too, and overt cultural markers create friction
4. **Avoid the "dark cosmic minimal" cluster** — Co-Star, The Pattern, Sanctuary, Nebula, TimePassages. Black background + simple white geometric form is over-occupied in our category
5. **Apple App Review Guideline 4.3 (spam family)** — 8 apps from one developer is already adjacent to scrutiny territory. Icons must look like 8 different products, not one product reskinned 8 times

### Technical constraints

- App Store master: 1024×1024 PNG, sRGB IEC61966-2.1, no alpha
- Per-app full export set (see Deliverables below)
- Must scale legibly from 20×20 (notification badge) to 1024×1024 (App Store)

## What we've already explored (and rejected)

So you don't re-tread:

| Direction | Why rejected |
|---|---|
| Vector-recreated metallic sphere (SVG) | Lacks photographic depth; quality ceiling too low |
| Same sphere + moon-phase lighting variations | Reads as Co-Star derivative; per-app differentiation too subtle at thumbnail scale |
| 朱砂 red seal (印章) + classical 篆体 character | Too explicitly Chinese for US-market positioning |
| Same background + small corner seal | Same cultural issue + Apple 4.3 spam risk |
| Sphere + different primary forms via DALL-E 3 | Cross-generation consistency too poor; doesn't reach premium feel |

We're not closed to revisiting any of these if you have a fresh angle, but please don't propose any of them as-is.

The catchlight has multiple tonal stops (bright white core → silver → gray → black), reading as a premium 3D-rendered metallic surface. High contrast, moody, contemplative.

  - Do / don't examples
  - Export specs
- **Source files**:
  - Figma (preferred master)
  - Adobe Illustrator (if used)
  - 3D scene files if applicable (Blender / Spline / Cinema 4D)
