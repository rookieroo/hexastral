插画 BRIEF

  • 风格锚点:国风二次元 / 新中式工笔(参考《阴阳师》《原神》立绘的线 + 渲染 + 描金,但更克制、更"器物/天象"少"人设")。
  • 母题(锁定,别换成 generic 月亮):八字=北斗紫微 · 姻缘=双鱼同心 · 黄历=日月祥云 · 面相=侧颜观相 · 风水=青绿山水 · 六爻=太极八卦 · 解梦=庄周梦蝶 · 梅花数=梅枝。
  • 色:深夜地子 + 月白 / 描金 / 朱砂点睛 / 黛青;每图主色≤3,全套统一地子保家族。
  • 构图:主体居中安全区、留白足、可缩放到 16px 仍认得出轮廓;交付 1024 + 矢量描边。
