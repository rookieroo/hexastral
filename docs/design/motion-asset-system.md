# HexAstral · 动效与素材系统 (Motion & Asset System)

> 一套统一的「运动语言 + 素材」规范，让 8 个 App 像**一个品牌**。
> 落在 `core-ui`，各 App 用 token 换**母题 + 分类印**即可，结构/参数/手感不动。
> 技术：定稿走 `reanimated v4 + expo-haptics`；重墨纹理预烘焙成 Rive / Lottie / sprite。HTML POC 只用来**锁手感**，不是实现。

相关：母题/印章 AI 提示词 [ai-image-prompts.md](ai-image-prompts.md)；图标定稿 = V15（暖纸 + 水墨暗月 + 朱印）；手感样机 `motion-poc-fate-flow.html`（含全部 5 件套）、`motion-poc-collection.html`（集章）、`motion-poc-loading-set.html`（loading 备选探索）。

## 1. 定稿五件套（8 App 复用 · 已在 fate-flow 锁手感）

> 这 5 个是**全局骨架**，每个 App 一字不改地复用结构与参数，只换**母题 + 分类印 + 文案**。
> 参数取自 `motion-poc-fate-flow.html`（`--s` = 慢放倍率，真机取 1）。

### 1.1 进场 · Logo 神奇移动（Magic Move / FLIP）

- **是什么**：Splash 居中的 **Logo（月）** 飞缩到首屏 **Hero 同一颗月** 的位置；纸底渐隐成夜、首屏文案在月周围上浮。前后是**同一颗月**，只是位/底变 → "神奇移动"。
- **关键定则**
  - 飞行目标 = **Hero 大月**（不是 header）。共享元素 = 月本身。
  - Splash **不套白底圆角卡**，直接显示 Logo（月）。
  - **header 首屏不放品牌/返回**，只留**分享**（首屏无上一级）。
  - Hero 月**排除出入场上浮**（`.bigmoon{animation:none}`），否则落位后会"二次闪"。其余文案照常上浮。
  - **交接干净**：到点瞬间露真月（与飞行体末态完全一致），次帧撤飞行体——**不要淡出重叠**（重叠是抖动来源）。
- **参数**：FLIP（测起止 rect 算 translate+scale）；`dur = clamp(580×s, 500, 1000)ms`（慢放也封顶，防"爬到终点"）；`ease = cubic-bezier(.4,0,.2,1)`（干净减速、无尾段急刹）。
- **RN**：`react-navigation` shared element / `react-native-reanimated` 的 `measure()` + `withTiming`；或 Skia 同元素跨屏。

### 1.2 Loading · 真实月相 + 步骤文案

- **是什么**：一颗水墨月走**真实月相**（新月→盈→满→亏→新月）；**暗面 = 图标的暗月**（同 v15 渐变 + 墨边）；**新月态 = App 图标本身**。下方轮播**正在做的事**。八 App 共用这套月相语言（中心母题可按 App 换，见 §3）。
- **结构**：受光面 `mphFace`（暖月底）＋ **阴影盘**（大圆 `r=48` 整体位移，`v15Shadow` 渐变中心全暗、外缘溶解为透明粒子）＋ **三频墨侵** `inkTerm`（大墨舌→纤维→飞溅粒子）＋ 圆形 `clip` ＋ 极淡颗粒 overlay。**无外圈描边**（夜空无 border，只极淡月晕）。
- **运动**：阴影盘 `cx` 随相位扫过 → 盈亏有据。`SWEEP=180`：盘**恰好**扫离月面，满月一闪而过，**几乎无"全白"空档**；`MOON_SPEED≈0.0027/帧`（一圈 ~6–7s，真机）。
- **步骤文案（关键·去神秘化）**：**不闪烁**；多端短语**淡入淡出轮流**（fade 0.42s，~1.6s 一条）：
  `校准真太阳时 → 起四柱 → 对齐天干 → 对齐地支 → 称五行旺衰 → 排布大运 → 勘合现运`（EN: Calibrating solar time → Casting Four Pillars → Aligning Heavenly Stems / Earthly Branches → Weighing Five Elements → Arranging Luck Pillars → Matching this phase）。**各 App 换成自己的计算步骤**。
- **短等待（换页/小请求）**：同一颗月的**小尺寸**（~64px）覆层，无文案；与全屏 loading 同源同语言。
- **RN**：Skia（位移盘 + noise displacement）或预渲染相位序列帧 Lottie；步骤文案纯文本轮播。

### 1.3 慢 · 墨晕转场（通用换屏 · 最高 ROI）

- **是什么**：从**点击点**起，一团墨化开揭示下一屏；揭示锋是**分层粒子墨晕**（同 loading 暗面语言）。两屏之间通用。
- **结构**：SVG `mask` 揭示圆，从点击点放大；圆填**渐隐渐变**（实心白核→边缘透明 = 浓→淡层次）＋ **双频位移**（低频大墨舌 + 高频飞溅粒子）→ 前缘是柔散粒子墨晕、非硬圆。**纸命书在夜屏上化开 = 高对比可见**（暗 mode 下尤其要让"被揭示的是纸/亮屏"）。
- **节奏**：**慢**——`dur≈2400ms`（真机即"原 0.3×"手感；常规速度太快、失仪式感）。
- **铁律**：墨晕转场 与 盖印**不可相邻**；若同屏，**盖的戳先隐**，待转场落定再触发盖印。
- **RN**：Skia mask + noise，或预渲染墨晕序列帧 Lottie；触发圆心 = 手势坐标。

### 1.4 用户盖印（落款 · 信/疑）

- **是什么**：读完命书，用户**主动**点「已阅 · 信」(认可) 或「存疑 · 疑」→ 水墨石章 **lilong** 落下 → 压 → 抬 → 宣纸右下显**小钤印**。盖印**由用户发起**，表达态度（不是系统自动）。
- **关键定则**
  - **落点 = 印记位**：石章水平按**印身重心**（非图框中心）对齐印记；基线落在印记底，盖压时石身全覆印面。
  - **印小**：成印 ~50px 的**落款**钤进宣纸右下留白（不是大贴纸）；石章随之收小（~132px）。
  - 章 = 2D 水墨**不形变**；冲击靠**投影收紧 + 朱砂 bleed + 触觉 + 纸面微震**。
  - 印面字 ∈ {命/相/卜}（分支）或 {信/疑}（态度）；各 App 石章造型不同。
- **参数**：`chopDrop ≈1200ms`（落 28% 现身 → 40% 触底 → 47% 微压 squash → 抬起 → 离场）；reveal/ bleed/ shadow 同拍；haptic：信 `impact 26`，疑 `[16,40,16]`。
- **POC**：`motion-poc-seal-stamp-real.html`（真章证明）、fate-flow 内联。

## 2. 运动语言原则（全局）

**缓动梯**

| 用途 | 曲线 | 时长(真机) |
|---|---|---|
| 转场/移动（干净到位） | `cubic-bezier(.4,0,.2,1)` | 0.5–0.6s |
| 墨晕化开（慢仪式） | `cubic-bezier(.3,.45,.2,1)` | ~2.4s |
| 盖印冲击（带微压） | `cubic-bezier(.3,.7,.25,1)` | ~1.2s |
| 文案淡入淡出 | ease | 0.4s（停留 ~1.6s） |
| 月相位移 | linear（连续） | 一圈 ~6–7s |

**铁则**：① 慢而有重量，服务"仪式/郑重"，不炫技；② 朱砂唯一强调、稀少即贵；③ 月相是 loading 的**唯一**通用语言（玄学指代清晰）；④ **不闪烁**——loading 用"正在做的事"短语轮播，给可信/及时感、去神秘；⑤ 墨晕转场 ≠ 盖印相邻；⑥ 物理动作必配 haptic。

**性能**：`feTurbulence` 很贵——逐帧多重 turbulence 仅限**单个小元素**（月/印边）；大面积（全屏墨晕）用**双频上限**或预烘焙噪声贴图 / Skia noise / Lottie 序列帧。HTML 里同时 >~20 个 filter 实例会冻渲染。

## 4. 集章留存系统（Collection）

**概念**：印章既是落款，也是**可收集资产**——各 App 盖章/签到，集齐入「印谱」→ 留存 + 收集欲。

**三条硬要求**：① **素材足够多**（命/相/卜 三字会重复，靠**石章造型**做变化，起步 12–24 方，分基础 + 节气限定）；② **风格统一**（全走同一条 img2img 转化词，见 §6）；③ **造型不同**（兽/螭龙/瓦/桥/天然钮，石/玉/玛瑙多色）。

**形态**：宣纸「印谱」册 → N 格印位；已集=显朱印(+石章)，未集=虚线位。签到/完成报告 → 盖下一枚 + 进度 `N/总`；集满触发彩蛋（母品牌印 / 限定）。POC：`motion-poc-collection.html`。

## 5. 尚未覆盖的场景（待补 · 已按 ROI 排序）

> 5 件套是骨架；下面是真实产品里**还会用到但 POC 未覆盖**的动效。给出"用什么墨语言做"，待逐个开 POC。**复用上面的语言，别新造方言。**

**A 类 · 高 ROI（建议优先补）**

1. **图表/数值 绘制 (chart & number draw-on)** — 命盘 astrolabe 与 大运 timeline 现在是**静态**，应**一笔描出**（spokes/轨迹 `stroke-dasharray` 描）、星点用**朱砂点批**逐个落、五行分数**count-up + 墨条填充**。报告是核心面，最值得。
2. **骨架屏 / 局部加载 (skeleton)** — 全屏等待用月相 loader；**局部/分段**加载需骨架。做法：宣纸上**淡墨条** + **一道飞白扫光**（非通用线性 shimmer）。
3. **空状态 (empty)** — 首次/无数据：空宣纸 + 极淡**母题水印**（暗月/卦）+ 一句引导 + 朱砂 CTA；月**淡而待**。
4. **错误/失败 (error)** — 请求失败/断网：**墨未聚成**（墨滴散开未成形）或**印盖歪/未盖实**（faint/broken seal）+ 重试 CTA。克制，不报红。
5. **付费解锁 (unseal / paywall)** — 业务关键、最扣母题：**封印 → 朱印"启封"裂开 / 墨幕揭开 → 隐藏命书显出**。可复用墨晕转场（局部）。
6. **下拉刷新 (pull-to-refresh)** — 日用刚需：下拉→**笔尖蘸墨/墨珠聚**；释放→**月相重新推演 / 新签墨晕显出**（接 loader 语言）。

**B 类 · 中 ROI**

7. **分享卡生成 (share compose)** — 增长闭环：hero + 判词 + 印 **逐层 ink-in** → 末了**盖印封缄**（防搬运水印）。
8. **抽屉/详情 (bottom sheet / drill-down)** — 点 大运 节点 / 命盘 星点 → 详情。做法：**纸笺自下抽出**，或从点击点**局部墨晕化开**（墨晕转场缩小版）。
9. **成功/庆祝 (集章满 / 里程碑)** — **不撒花**；克制的**朱砂晕开 + 母品牌印盖下**。
10. **Tab / 分支切换** — 比全屏墨晕更轻：**轻墨晕** 或 对应分类印**一闪/轻盖** + crossfade。

**C 类 · 低 ROI / 点缀**

11. **吐司/内联提示 (toast)** — 一行墨字自纸面浮起/淡出，朱点前缀。
12. **按压/微交互 (press states)** — CTA 按下朱砂微缩 + haptic；pills 选中。
13. **滚动揭示 (scroll reveal)** — 命书长文 section 入视口时**飞白揭示 / 淡入上浮**。
14. **生辰输入 (birth time wheel)** — 干支/时辰滚轮（偏 onboarding，本期非重点）。

## 6. 素材库 + 采集 / 转化流水线

**素材词汇表**：宣纸 · 墨(飞白/墨晕/一笔) · 朱砂 · 月相 · 篆字 · 碑拓金文 · 卦爻 · 星轨命盘 · 罗盘 · 铜钱 · 墨蝶 · 山水侧脸 · 红线(缘) · 气脉 · enso 圆相 · **印章**。

**继续采集（用于风格化）**：
- **印章为主**（集章需量）：多材质（寿山/青田/鸡血/青玉/白玉/玛瑙）、多钮（兽/螭龙/瓦/桥/天然）、多色。目标 12–24+ 方。
- 文房延伸（静态/点缀）：印泥盒、毛笔、墨锭砚台、古钱、卦签、罗盘。
- 采集规格：单物居中、纯净底（深或纯灰最易抠）、光向一致、尽量不裁全身。

**统一转化（关键）**：**只用一条** img2img 提示词跑全部素材（见 [ai-image-prompts.md](ai-image-prompts.md) 「印章实物 → 水墨风 2D」），只换 `石/玉` 材质尾巴 + 锁形 + **补全被裁体形** → 整套**风格统一、造型不同**。

**落盘**：`docs/design/raw/seals/`（朱印 命/相/卜、信/疑）；石章风格稿建议 `docs/design/assets/chops/{material}-{form}-NN.png`。

## 7. 生产端实现映射（HTML POC → RN）

| POC 手法 | 真机实现 | 备注 |
|---|---|---|
| FLIP（measure rect + transform） | reanimated `measure()` + `withTiming` / 共享元素转场 | 进场 magic-move |
| SVG mask + `feTurbulence` 位移 | **Skia**（mask + noise displacement）或**预渲染序列帧 Lottie** | 月相 / 墨晕转场（大面积别逐帧 turbulence） |
| `mix-blend-mode: multiply`（石章去白边） | 预抠透明 PNG / Skia blend | 石章素材应出**透明底**，免 multiply |
| `getAnimations()` seek（验证） | — | 仅 POC 调试 |
| 颗粒/墨纹 filter | **预烘焙噪声贴图** | 省 GPU |
| haptic | `expo-haptics` `impactAsync` | 盖印/抛钱/摇卦 |

**纹理一律预烘焙**：照片级墨丝/粒子靠**烘好的 sprite / Rive / Lottie**，不靠真机逐帧 `feTurbulence`（贵）。几何与节奏照本规格，换实现不换观感。

## 8. 原则 + 优先级

**原则**：① 一套墨物理引擎全局复用；② 朱砂唯一强调、稀少即珍贵；③ 纸/墨/朱三层语义恒定；④ core-ui + token 主题化（换母题/印不换结构）；⑤ 克制、慢、有重量；⑥ loading **去神秘化**（步骤文案）、不闪烁。

**优先级**：① 五件套（已锁手感）→ ② §5-A 图表绘制 / 骨架 / 空 / 错误 / 解锁 / 下拉（补全真实产品闭环）→ ③ 集章（留存）→ ④ coin-cast 抛钱→爻（核心仪式）→ 其余按需。
